## Slide 2: The Problem

### Chat interfaces create tunnel thinking.

```
┌─────────────────────────────────────────┐
│  Traditional Chat                        │
│                                          │
│  User: Question                          │
│  AI: Answer                              │
│  User: Follow-up                         │
│  AI: Answer                              │
│  User: Wait, go back...                  │
│  AI: ???                                 │
│  User: *starts new chat*                 │
│                                          │
│  ❌ Linear                               │
│  ❌ No branching                         │
│  ❌ Context lost                         │
│  ❌ Can't compare alternatives           │
└─────────────────────────────────────────┘
```

**$20B+ spent on AI APIs annually.**
**Zero tools for structured AI thinking.**

## Slide 3: The Solution

### Tension: Graph-based AI interaction.

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│    ┌──────┐                                         │
│    │ Root │──┬──► [Approach A] ──► [Deep 1] ──► ... │
│    │      │  │                                      │
│    └──────┘  ├──► [Approach B] ──► [Deep 1]        │
│              │                                      │
│              └──► [Approach C] ──► [Dead end]      │
│                                                     │
│    ✅ Spatial                                       │
│    ✅ Branching                                     │
│    ✅ Persistent                                    │
│    ✅ Comparable                                    │
└─────────────────────────────────────────────────────┘
```

**Dialog = Graph, not Thread.**
